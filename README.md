My personal website.

## Built With

* [Gatsby.js](https://www.gatsbyjs.org/)
* [React](https://reactjs.org/)

## Development

1. `cd gatsby`
2. `npm ci`
3. `npm start`

## Deployment

All pushes to main will trigger a Netlify build of the Gatsby site, if needed.
